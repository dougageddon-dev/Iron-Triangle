# Pick Two

> The Iron Triangle — you can't have it all.

A simple interactive web app based on the classic project management principle: **Cheap**, **Fast**, and **Quality** — pick any two.

## Features
- Smooth entrance animations
- Auto-deselects a random option when you pick a third
- Background glow blends the colors of your two selections
- Light / dark theme toggle
- Installable as a PWA on iOS and Android
